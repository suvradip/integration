Event name: exportCancelled

Triggered when the default behavior of the beforeExport event is cancelled by calling the eventObj.preventDefault() method.

All events, when triggered, will provide two parameters for the handler - eventObj (object containing information generic to all events) and dataObj (object containing information specific to an event).

Click the Export Chart button, in the top-right corner of the chart ( you may have to increase the width of the Result pane to access the button ). From the menu, select the format in which the chart has to be exported.

This triggers the beforeExport event, which calles the eventObj.preventDefault() method. This, in turn, triggers the exportCancelled event.

Scroll down to the table rendered below the chart to view information contained in the dataObj object. To view information contained in the eventObj object, open the console.
