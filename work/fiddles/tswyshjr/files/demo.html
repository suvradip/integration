Event name: renderCancelled

Triggered when the default behavior of the beforeRender event is cancelled using the eventObj.preventDefault() method.

All events, when triggered, will provide two parameters for the handler - eventObj (object containing information generic to all events) and dataObj (object containing information specific to an event).

Scroll down to the table rendered to view information contained in the dataObj object. To view information contained in the eventObj object, open the console.
