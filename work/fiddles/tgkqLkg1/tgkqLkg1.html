Method name: removeEventListener() (static)

Removes the event listener(s) bound to an event configured for all charts on the page.

For this implementation, we have three pie charts rendered on the same page. When Add event listener for all charts is clicked, the dataplotRollOver event is configured for all three pie charts.
When Remove event listener for all charts
is clicked, the event listener is removed for all charts rendered on the page.
