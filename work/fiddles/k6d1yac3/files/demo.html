Method Name: dataReady()

Determines whether a chart will render properly with the data set on it. This includes data that is set using the setChartData() or setChartDataUrl() methods.

For this implementation, we have a column 2D chart and a drop-down from which the user selects the data to plot on the chart. As the chart used is a single-series column 2D chart, the method is configured to show a Data invalid message every time multi-series data is passed to the chart.
