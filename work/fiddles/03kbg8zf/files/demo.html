Event name: pinned

Triggered in the pin mode of a zoom-line chart, when the user performs data selection to pin a range.

All events, when triggered, will provide two parameters for the handler - eventObj (object containing information generic to all events) and dataObj (object containing information specific to an event).

Switch to the pin mode (click the switch button in the top-right corner of the chart). Drag the mouse pointer over a set of data points on the chart to pin the data and trigger the event. Scroll down to the table rendered below the chart to view information contained in the dataObj object. To view information contained in the eventObj object, open the console.
