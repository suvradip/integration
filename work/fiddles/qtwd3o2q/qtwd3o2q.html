Method Name: configure()

Configures status messages that are displayed while rendering a chart.

For this implementation, the configure() method is used with the dataEmptyMessage option.
To show how this method works, no data - or empty data - is passed to the doughnut 2D chart.
Not data to display. is the default message rendered when empty data is passed to the chart. Using the configure() method, this message has been changed to No data to load. Please check the data source.
