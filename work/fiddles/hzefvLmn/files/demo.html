Event name: chartMouseMove

Triggered when the mouse pointer is moved over a chart. To enable this event for a chart, set the enableChartMouseMoveEvent attribute to 1 .

All events, when triggered, will provide two parameters for the handler - eventObj (object containing information generic to all events) and dataObj (object containing information specific to an event).

Move the mouse pointer over the chart to trigger the chartMouseMove event. Scroll down to the table rendered below the chart to view information contained in the dataObj object. To view information contained in the eventObj object, open the console.
