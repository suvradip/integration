Event name: dataSubmitError

Triggered when there is an ajax error in sending the POST request with the user-submitted data in interactive charts (select-scatter, drag-node, and dragable charts).

All events, when triggered, will provide two parameters for the handler - eventObj (object containing information generic to all events) and dataObj (object containing information specific to an event).

Update the drag-node chart (add, update, or delete any of the chart elements) and click Save . The dataSubmitError event is triggered when an ajax error is encountered while submitting data.

Scroll down to the table rendered below the chart to view information contained in the dataObj object. To view information contained in the eventObj object, open the console.
