Method Name: clone()

Creates a copy of a chart instance, creating a new chart with identical construction properties of the chart being cloned. The cloned chart, assigned an auto-generated ID, is rendered in a container DOM element that is explicitly provided.

Click the Create an exact clone button to create an exact replica of the column 2D chart shown below. Click the Create a modified clone button to create a bar chart replica of the column chart.

Scroll down to see the cloned chart rendered below the original column chart.
