Event name: dataSubmitCancelled

Triggered when the default behavior of the beforeDataSubmit event is cancelled by calling the eventObj. preventDefault() method.

All events, when triggered, will provide two parameters for the handler - eventObj (object containing information generic to all events) and dataObj (object containing information specific to an event).

Update the drag-node chart (add, update, or delete any of the chart elements) and click Save . This triggers the beforeDataSubmit event, which calls the eventObj.preventDefault() method. This, in turn, triggers the dataSubmitCancelled event.

Scroll down to the table rendered below the chart to view information contained in the dataObj object. To view information contained in the eventObj object, open the console.
