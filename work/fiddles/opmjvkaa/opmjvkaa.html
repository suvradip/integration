Method Name: setChartAttribute()

Updates a chart’s attributes with a new attribute-value pair, thus updating the chart’s data definition root (the chart object in the JSON data). This function is useful when a chart’s configuration has to be updated after it has been rendered once. To read more about this method, click here.

From the drop-down below, when a theme is selected, the setChartAttribute() method is used to update the value of the theme attribute. The chart is then rendered with the selected theme.
