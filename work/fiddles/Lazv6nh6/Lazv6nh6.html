Event name: dataSubmitted

Triggered when the ajax POST request for sending user-submitted data in interactive charts (select-scatter, drag-node, and dragable charts) to the URL is successfully completed.

All events, when triggered, will provide two parameters for the handler - eventObj (object containing information generic to all events) and dataObj (object containing information specific to an event).

Update the drag-node chart (add, update, or delete any of the chart elements) and click Save . The dataSubmitted event is triggered when the updated data is successfully submitted.

Scroll down to the table rendered below the chart to view information contained in the dataObj object. To view information contained in the eventObj object, open the console.
