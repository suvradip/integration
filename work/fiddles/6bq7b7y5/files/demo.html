Method Name: removeEventListener()

Removes the event listener(s) bound to an event using the addEventListener() method.

For the pie chart below, first click the Set Event Listener button to add a listener for the dataplotRollOver event. Then, click the Remove Event Listener button to remove the listener for the dataplotRollOver event.
After the listener has been removed, the chart details are not shown when the mouse pointer is rolled over a pie slice.
